Getting Started:

To use ShaderOne, On material select the shader under echoLogin > ShaderOne which has the default settings.

To modify shader with new features you need to goto Window > ShaderOne Generator
Choose which options you need and press Generate.

See the Documentation PDF’s in echoLogin > ShaderOne > !Docs for more information on usage.

Make sure you save a backup of your settings from the Generator !

To run the demos read the ReadMe.txt file in each demo folder.
